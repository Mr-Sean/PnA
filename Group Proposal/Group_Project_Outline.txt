Project Title: Poll Site
Project Stack: MERN
GitHub Repo Link: https://github.com/BenjaminSeang/Poll-Site

Project Description: 
An online polling/rating website which allows users to vote on certian topics or initiate their own polls/ratings. For example, a user can initiate a poll "Which is the best cafe in the 5th street in San Jose," and give a list of options. Other logged in users will be able to see and vote on this poll.
